\# FrangoPDV



Sistema simples de venda e impressão para comércio de frango.



\## Objetivo



Criar um PDV simples para:

\- vendas rápidas

\- impressão térmica

\- comprovante

\- controle básico



\## Tecnologias



\### Frontend

\- React

\- Vite



\### Backend (futuro)

\- Flask

\- SQLite



\## Status atual



Frontend inicial funcionando com:

\- tela de venda

\- adicionar itens

\- total automático

\- impressão básica
